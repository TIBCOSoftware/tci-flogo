removeProvider: Type<IContribRemoveProvider>
  • Defined in app/contrib/wi-contrib.ts:305
description

Optional remove handler for handling cleanup actions on external domains, invoked when the contribution instance is deleted. The key information for delete comes from the context itself.

example
