Optional array

array: boolean
  • Defined in common/models/contrib.ts:573

Indicates if the argument should be an array. False by default. This field in combination with type determine the expected parameter type.

example

{ name: "example", type: "string", array: true } means function expects a parameter of type string[] or Array

Optional variable

variable: boolean
  • Defined in common/models/contrib.ts:592

Indicates that the function can handle an indefinite number of arguments starting in this argument's position. False by default. Only allowed for trailing arguments.

This flag works in combination with the "required" flag in the following way: variable=true and required=true => argument expected one or more times variable=true and required=false => argument expected zero or more times

default

false
