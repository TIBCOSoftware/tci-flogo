getAuthLoginURL

  • getAuthLoginURL(): string
    • Defined in common/models/contrib.ts:1669
    desc

    Returns the Authentication Login URL provided by the Connector without any state information parameters

    example

    https://login.salesforce.com/services/oauth2/authorize?response_type=code&client_id=CLIENTID&redirect_uri=REDIRECTURL&display=popup&prompt=login consent where REDIRECT_URL=https://localdev-integration.cloud.tibco.com/wistudio/connectiondetails NOTE: state information parameter for Salesforce "state" is not part of the URL

    Returns string
