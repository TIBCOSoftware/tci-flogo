Interface IAuthenticationResult

Represents the Authentication Result data structure returned by the connector contribution developer when a OAUTH login action is invoked from the connector form buttons defined in connector.json

The {{IAuthenticationResult}} is returned with the {{IActionResult}} result parameter.
