getAuthData

  • getAuthData(): IOAuthData | IBasicAuthData
    • Defined in common/models/contrib.ts:775

    Returns the Authentication Data Object {{IOAuthData}} which holds the information for {AUTHENTICATION_TYPE.BASIC} OR Returns the Authentication Data Object {{IOAuthData}} which holds the information for {AUTHENTICATION_TYPE.OAUTH1} | {AUTHENTICATION_TYPE.OAUTH2} |{AUTHENTICATION_TYPE.OAUTH2_TWOLEG} type authentication mechanisms. The contribution developer needs to provide this information based on which the contribution mechanism will introspect the callback return state information. It is not assumed that every authentication type will not have this information requirement as it may not involve a callback. In case of OAUTH type authentication it is also not assumed that the state information is available by introspecting a fixed parameter name e.g. "state" so the contribution developer need to provide this based on the SAAS authentication provider specifications.

    Returns IOAuthData | IBasicAuthData

    | {IBasicAuthData}
